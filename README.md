# Portfolio

My personal portfolio website — a showcase of the projects I've built while studying Software Engineering at VIA University College.

## Why I Built This

I wanted a place that felt like *me* rather than a generic template. Instead of a standard list of links, I built an interactive 3D carousel and a cinematic fullscreen grid to display my work in a way that actually feels engaging to navigate.
